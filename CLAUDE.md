# Claude AI 開発指示

## 言語設定

このプロジェクトでは、AI とのやりとりを**日本語**で行います。

- **対話・説明**: すべての会話、説明、計画は日本語で記述してください
- **コードコメント**: Rust コード内のコメントは日本語で記述してください
- **ドキュメント**: README、設計書、仕様書などすべてのドキュメントは日本語で記述してください
- **コミットメッセージ**: Git コミットメッセージも日本語で記述してください
- **実装計画**: AIエージェントのplan機能で作成される実装計画書も全て日本語で記述してください

## プロジェクト概要

**GLM (GitHub Life Manager)** は、GitHub の Issue を TUI (Terminal User Interface) で管理する Rust 製のツールです。

### 技術スタック

- **言語**: Rust (Edition 2021)
- **TUI フレームワーク**: Ratatui
- **GitHub API クライアント**: Octocrab
- **非同期ランタイム**: Tokio

### 主要機能

- GitHub Issue の閲覧と編集
- リポジトリ選択機能
- Issue 作成機能
- キーボードベースの操作

## コーディング規約

### コメントスタイル

```rust
// 単一行コメントは日本語で記述
// 関数やロジックの説明を明確に

/// ドキュメントコメントも日本語で記述
/// この関数は GitHub API からリポジトリ一覧を取得します
fn fetch_repositories() -> Result<Vec<Repository>> {
    // 実装
}
```

### エラーハンドリング

- `anyhow` クレートを使用してエラーコンテキストを提供
- エラーメッセージは日本語で分かりやすく

```rust
.context("リポジトリの取得に失敗しました")?
```

### 命名規則

- 変数名、関数名、型名は英語を使用（Rust の慣習に従う）
- ただし、ビジネスロジックに関する説明コメントは日本語で記述

## UI/UX 設計方針

### キーボードショートカット

- `j` / `↓`: 下に移動
- `k` / `↑`: 上に移動
- `e`: 編集
- `r`: リポジトリ選択
- `n`: 新規 Issue 作成
- `q`: 終了
- `Esc`: 戻る

### レイアウト

- 左パネル: リスト表示（30%）
- 右パネル: 詳細表示（70%）
- ステータスバー: ヘルプテキストとコンテキスト情報

## 開発ワークフロー

### 変更を加える際の手順

1. **理解**: 既存のコードとアーキテクチャを理解
2. **設計**: 変更内容を日本語で説明
3. **実装**: Rust のベストプラクティスに従って実装
4. **テスト**: 手動テストで動作確認
5. **ドキュメント**: 必要に応じてドキュメントを更新

### Git ワークフロー

- ブランチ名: 日本語可（例: `feature/リポジトリ選択機能`）またはケバブケース（例: `feature/repo-selection`）
- コミットメッセージ: 日本語で明確に記述

## API 設計方針

### GitHub API 利用

- 認証: `gh auth token` コマンドから取得
- レート制限: エラーハンドリングでユーザーフレンドリーなメッセージを表示
- エンドポイント: Octocrab の API を活用

### エラーメッセージ

ユーザーに表示するエラーメッセージは必ず日本語で：

```rust
eprintln!("エラー: GitHub API への接続に失敗しました");
```

## テスト方針

現時点では手動テストを中心に実施：

- 各機能の正常系フロー
- エラーハンドリング（ネットワーク障害、認証エラー等）
- エッジケース（空のリスト、大量データ等）

## パフォーマンス考慮事項

- API 呼び出しは非同期で実行
- UI の描画は可能な限りブロックしない
- 大量データの場合はページネーションを検討

## セキュリティ

- GitHub トークンは環境変数やシステムから取得（ハードコードしない）
- ユーザーデータは適切に扱い、ログに出力しない

## その他の注意事項

- 既存のコードスタイルを尊重し、一貫性を保つ
- 不要な依存関係の追加は避ける
- プロジェクトのシンプルさを維持する
